# Translator App Project

## 📃 Description
The Translator App is a project developed in Node.js, using SQLite for database management and the DeepL Free API for translation services.
The application allows users to translate text between multiple languages, store translation history in both the database and local storage, and provides an easy-to-use interface for seamless interaction.

## 🛠 Technologies Used
- **Node.js**: for server-side logic
- **SQLite**: for database management
- **DeepL Free API**: for translation services
- **HTML/CSS**: for user interface

## ✨ Features
- Multiple language support for translation
- Integration with DeepL API for accurate translations
- User-friendly interface for seamless interaction
- Translation history stored in both **local storage** (for immediate access) and **SQLite database** (for permanent storage)
